# Global Country Stats Dashboard

A static dashboard for comparing country-level population, GDP per capita, life expectancy, internet access, and CO₂ emissions.

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Test

```bash
npm test
```

## Data

The dashboard currently uses a small sample dataset in `data/countries.js` for development. Replace it with validated World Bank, UN, or OECD feeds before using the numbers for formal analysis.
